## ğŸ” **Overview**

This project examines how four audio environments influence study focus:

- **Silence (no audio)**
- **Instrumental Music (lo-fi and kawaii-style study tracks)**
- **Lecture Audio (recorded class lecture videos)**
- **Music with Lyrics (primarily K-pop and pop music)**

Over **14 days**, I recorded **87 study sessions**, tracking:

- Start and end time  
- Session duration  
- Activity  
- Audio environment  
- Session observations  
- A **1â€“5 focus rating**

The study initially followed my natural listening habits.  
Midway, I realized the environments were unevenly represented, so I deliberately added more **silence** and **instrumental** sessions to balance the dataset.

---

## ğŸ¯ **Key Findings**

- **Silence** and **Instrumental music** produced the highest focus.  
- **Lecture Audio** and **Music with Lyrics** lowered concentration.  
- **Duration did not predict focus** â€” longer sessions did not increase focus.  
- **One-way ANOVA (p < 0.001)** confirmed statistically significant differences across environments.  
- **Tukey post-hoc tests** identified which environments differed from each other.
